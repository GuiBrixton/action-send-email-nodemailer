# action-send-email-nodemailer

This action allows to Send an Email via Nodemailer.

## Inputs

## `to`

**Required** The Receiver

## `from`

**Required** The Sender

## `subject`

**Required** Email subject

## `body`

**Required** Email body

## Outputs

## `time`

The time we greeted you.

## Example usage

uses: actions/hello-world-javascript-action@v1.1 with:
who-to-greet: 'Mona the Octocat'
